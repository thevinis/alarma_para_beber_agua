este é um app em kotlin, para da um alarta a cada hora determinada pelo usuario pela interface, e tem a opçao de definir quanto de agua (ml) voce vai fica bendo, quando da o horario definido,
por exemplo uma hora depois um popup aparece e voce clica nele e registra o consumo ou aperta ignorar, ele toca quando fora do silencioso e vibra no silencioso.
